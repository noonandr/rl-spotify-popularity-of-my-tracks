# Risk Ledger 
Display your Spotify tracks based on how popular they are


## Installation

cd into root folder:

```
$ cd rl-spotify-popularity-of-my-tracks
```

Install npm modules:


```
$ npm install
```

## Run Server

To run the server:

```
$ npm run serve
```
It will probably be running on http://localhost:8080/
